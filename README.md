# Ghetty Motor Home Questionnaire

Cuestionario interactivo para organizar el negocio de alquiler de motorhome.

## Página pública

Cuando GitHub Pages termine de desplegar, la página debe abrir aquí:

https://jaimeespinalpr.github.io/ghetty-motorhome-questionnaire/

## Envío por email

El formulario usa FormSubmit y envía las respuestas a:

jaimeespinalpr@gmail.com

La primera vez FormSubmit puede enviar un email de confirmación al correo destino. Después de confirmar, los envíos llegan automático.

## Qué incluye

- Logo de Ghetty Motor Home integrado.
- 50 preguntas interactivas.
- Opciones A, B y C.
- Campo para respuesta personalizada.
- Revisión final antes de enviar.
- Email con cada pregunta y solo la respuesta seleccionada o escrita.
